Preenchimento de datasets para treinamento de dados

Esta ferramenta tem como objetivo criar um dataset de perguntas e respostas relacionado a área de licitações públicas. Há um conjunto de arquivos PDF que são os exemplos de dados de treinamento para os documentos ETP (Estudo Técnico Preliminar) e TR (Termo de Referência). Baseado nesses arquivos, o especialista técnico nessas áreas de licitações deve elaborar perguntas para preencher o dataset, de modo que a resposta seja um recorte ou resumo do texto localizado em alguns dos PDFs.

É importante ressaltar que a pergunta deve ser bem elaborada, sem erros de português e de forma que tenha o máximo de detalhamento possível, a fim de que a resposta possa contemplar de forma coesa.
